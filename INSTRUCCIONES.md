# Cómo subir el monitor a GitHub

Sigue estos pasos en orden. Solo necesitas hacerlo una vez.

---

## Paso 1 — Descargar GitHub Desktop

Ve a https://desktop.github.com y descarga la app para Mac.
Ábrela e inicia sesión con tu cuenta (JavierIniguezz).

---

## Paso 2 — Clonar tu repositorio

En GitHub Desktop:
- Pulsa "Clone a Repository from the Internet"
- Selecciona "monitor-empleo" de tu lista
- Elige una carpeta en tu Mac (por ejemplo, Documentos)
- Pulsa "Clone"

Ahora tienes la carpeta del proyecto en tu Mac.

---

## Paso 3 — Copiar los archivos

Copia estos 4 archivos/carpetas a la carpeta del repositorio
(la que acabas de clonar):

  - monitor.py
  - config.yaml
  - requirements.txt
  - .github/  (la carpeta entera, con todo lo que hay dentro)

---

## Paso 4 — Subir a GitHub

En GitHub Desktop verás los archivos nuevos listados.
Escribe un mensaje en el campo "Summary", por ejemplo:
  "Añadir monitor de empleo"

Pulsa "Commit to main" y luego "Push origin".

---

## Paso 5 — Activar GitHub Pages

Ve a github.com/JavierIniguezz/monitor-empleo
- Pulsa Settings (arriba a la derecha del repositorio)
- En el menú lateral, pulsa Pages
- En "Branch" selecciona "main" y la carpeta "/ (root)"
- Pulsa Save

En unos minutos tu página estará en:
  https://javieriniguezz.github.io/monitor-empleo/

---

## Paso 6 — Primera ejecución manual

Para no esperar a mañana:
- Ve a la pestaña "Actions" en tu repositorio de GitHub
- Haz clic en "Monitor diario de empleo"
- Pulsa "Run workflow" → "Run workflow"

En 2-3 minutos el script habrá corrido y tu página estará actualizada.

---
